git init - inicilizar o repositório
git config user.name - define o nome de usuário
git config --global user.name - Configurar o projeto global
git config user.email - define email
git status - Verificar o status do arquivo criado.
git add . - Adicionar as alterações do arquivo ao repositório
git commit -m  - "Digite a descrição do seu commit"
git log - Verificar o log
Novos comandos.
git branch (verifica quais branchs existem e cria uma nova branch com o nome informado com -d deleta)
git checkout (troca de branch e com -b cria e troca de branch)
git switch (troca de branch e com -s cria e troca de branch)
git merge (mescla as informações de branch para outra)
